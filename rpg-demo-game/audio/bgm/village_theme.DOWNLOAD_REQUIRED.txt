- Bitrate: 128-192 kbps

## Installation
1. Download or create a peaceful village-themed music track
2. Convert to OGG format if needed (use Audacity or FFmpeg)
3. Save as: audio/bgm/village_theme.ogg
4. Delete this placeholder file

## Conversion Command (FFmpeg)
ffmpeg -i input_music.mp3 -c:a libvorbis -q:a 5 village_theme.ogg

## Alternative: Use RPG Maker MZ Default
If you don't have custom audio, the map will fallback to "Town1" (default RPG Maker MZ track).
Edit data/Map001.json and change "village_theme" back to "Town1" in the bgm.name field.
