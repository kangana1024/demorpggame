## Requirements
- Format: OGG Vorbis (.ogg)
- Bitrate: 128-192 kbps
- Style: Mysterious, tense, slightly dangerous
- Duration: 2-4 minutes (loopable)

## Installation
1. Download or create a mysterious forest-themed music track
2. Convert to OGG format if needed (use Audacity or FFmpeg)
3. Save as: audio/bgm/forest_theme.ogg
4. Delete this placeholder file

## Conversion Command (FFmpeg)
ffmpeg -i input_music.mp3 -c:a libvorbis -q:a 5 forest_theme.ogg

## Alternative: Use RPG Maker MZ Default
If you don't have custom audio, the map will fallback to "Field2" (default RPG Maker MZ track).
Edit data/Map002.json and change "forest_theme" back to "Field2" in the bgm.name field.
