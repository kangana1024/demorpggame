## Audio Conversion

If your source audio is not in OGG format:

```bash
# Install FFmpeg
# macOS: brew install ffmpeg
# Ubuntu: sudo apt install ffmpeg

# Convert MP3 to OGG (SE quality)
ffmpeg -i source.mp3 -c:a libvorbis -q:a 4 audio/se/quicksand_squelch.ogg

# Convert WAV to OGG
ffmpeg -i source.wav -c:a libvorbis -q:a 4 audio/se/quicksand_squelch.ogg
```

Quality setting `-q:a`:
- `3` = ~96 kbps (good for short SE)
- `4` = ~112 kbps (recommended)
- `5` = ~128 kbps (higher quality)
