## 1. License Type Reference

### Common License Types

| License | Full Name | Commercial Use | Attribution | Share-Alike | Modifications |
|---------|-----------|----------------|-------------|-------------|---------------|
| **CC0** | Creative Commons Zero | ✅ Yes | ❌ Not Required | ❌ No | ✅ Allowed |
| **CC-BY** | Creative Commons Attribution | ✅ Yes | ✅ Required | ❌ No | ✅ Allowed |
| **CC-BY-SA** | Creative Commons Attribution-ShareAlike | ✅ Yes | ✅ Required | ✅ Required | ✅ Allowed |
| **CC-BY-NC** | Creative Commons Attribution-NonCommercial | ❌ No | ✅ Required | ❌ No | ✅ Allowed |
| **CC-BY-NC-SA** | CC Attribution-NonCommercial-ShareAlike | ❌ No | ✅ Required | ✅ Required | ✅ Allowed |
| **SIL OFL** | SIL Open Font License | ✅ Yes | ✅ In credits | ❌ No | ✅ Allowed |

### What Each Requirement Means

| Term | Description |
|------|-------------|
| **Commercial Use** | Using the asset in a game that is sold or monetized |
| **Attribution** | Crediting the original creator in your game's credits or documentation |
| **Share-Alike** | Derivative works must use the same or compatible license |
| **Modifications** | Editing, resizing, recoloring, or adapting the original asset |

### How to Check itch.io Asset Licenses

1. Go to the asset pack page on itch.io
2. Scroll down to the **"More Information"** section
3. Look for:
   - "License: [License Type]"
   - Download the README or LICENSE file if included
4. If unclear, contact the creator directly via itch.io messaging

### ⚠️ Best Practices for itch.io Assets

- **Download and save license files** with each asset pack
- **Screenshot the license terms** from the asset page
- **Keep a record** of where each asset was obtained
- **When in doubt, ask** the creator before using commercially

### LPC (Liberated Pixel Cup) Assets

Many OpenGameArt assets are from the Liberated Pixel Cup contest and use **CC-BY-SA 3.0** or **dual license (CC-BY-SA 3.0 + GPLv3)**.

**CC-BY-SA 3.0 Attribution Requirements**:
- Credit the original author(s)
- Link to the original asset page if possible
- If you modify the asset, indicate changes made
- License your derivative work under CC-BY-SA 3.0 or compatible

**Attribution Example**:
```
[Asset Name] by [Author Name]
Licensed under CC-BY-SA 3.0
https://creativecommons.org/licenses/by-sa/3.0/
Source: [OpenGameArt URL]
```

### CC0 (Public Domain) - No Attribution Required

Assets marked CC0 can be used freely without attribution. However, it's good practice to credit creators when possible.

## 9. License Compatibility Matrix

### Can These Licenses Be Combined?

| License A | License B | Compatible? | Notes |
|-----------|-----------|-------------|-------|
| CC0 | Any | ✅ Yes | CC0 is maximally compatible |
| CC-BY | CC-BY | ✅ Yes | Just credit both |
| CC-BY | CC-BY-SA | ⚠️ Partial | Result must be CC-BY-SA |
| CC-BY-SA | CC-BY-SA | ✅ Yes | Keep same license |
| CC-BY-SA | CC-BY-NC | ❌ No | Incompatible (commercial vs non-commercial) |
| SIL OFL | Any | ✅ Yes | Fonts don't affect other assets |
| VisuStella TOS | Any | ✅ Yes | Plugin license is separate |

### Share-Alike Implications

If you use **CC-BY-SA** assets and modify them, your modifications must also be CC-BY-SA. However:
- This applies only to the modified asset, not your entire game
- Unmodified CC-BY-SA assets don't make your whole project CC-BY-SA
- Your game code and original content remain your own

## License Compliance Checklist

Before releasing your game, verify:

- [ ] All font licenses documented (SIL OFL for Thai fonts)
- [ ] VisuStella attribution in credits
- [ ] All audio sources credited appropriately
- [ ] OpenGameArt asset licenses verified and documented
- [ ] itch.io asset licenses verified (screenshotted/saved)
- [ ] No CC-BY-NC assets used in commercial release
- [ ] Credits file/screen includes all required attributions
- [ ] License files saved with downloaded assets
- [ ] No conflicting licenses in project

## Quick Reference Card

### Safe for Commercial Use (Usually)
- ✅ CC0 (Public Domain)
- ✅ CC-BY (with attribution)
- ✅ CC-BY-SA (with attribution, share-alike)
- ✅ SIL OFL (fonts, with attribution)
- ✅ VisuStella plugins (with attribution)
- ✅ Official RPG Maker DLC

### NOT Safe for Commercial Use
- ❌ CC-BY-NC (Non-Commercial)
- ❌ CC-BY-NC-SA (Non-Commercial Share-Alike)
- ❌ "Personal use only" assets
- ❌ Assets without clear license

### When in Doubt
1. Contact the asset creator
2. Use alternative CC0 or paid assets
3. Create your own asset
4. Consult a legal professional for commercial releases
